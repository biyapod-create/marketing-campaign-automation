## 🎯 QUICK ACTIONS

### ✅ BEFORE Adding New Company
1. Open: Duplicate Check sheet
2. Search: Company name, website, email
3. Search: Previous month files
4. If NOT found → OK to add
5. If FOUND → DO NOT CONTACT (unless 90+ days)

### ✅ AFTER Sending Email
Update 4 places:
1. Email Campaign Log → Add new row
2. Master Contact Database → Update "Total Emails Sent"
3. Follow-up Tracker → Schedule follow-up (Day 3)
4. Campaign Dashboard → Update "Emails Sent"

### ✅ WHEN Company Responds
Update 4 places:
1. Email Campaign Log → Mark "Replied", add date & summary
2. Master Contact Database → Change status to "Replied"
3. Follow-up Tracker → Mark completed, create new follow-up
4. Campaign Dashboard → Update "Replies"

### ✅ NEW MONTH SETUP (Example: Feb 1)
1. Copy January file → Rename to February
2. KEEP: All Master Contact Database entries
3. KEEP: All Duplicate Check entries
4. CLEAR: Email Campaign Log
5. CLEAR: Completed follow-ups
6. RESET: Campaign Dashboard to zero
